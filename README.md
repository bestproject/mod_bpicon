# mod_bpicon
A Joomla! 3 module for displaying an FontAwesome Icon

## Functionality
- Using FontAwesome Icons (easy selection from a preview window)
- Linking type (title, text, icon, all)
- Linking (external url, article, menu item)
- Update server
- Bootstrap3 ready
